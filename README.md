# How to use it?

1. Open `chrome`, then visit `web.whatsapp.com`
2. Press `F12`, click at `console`
3. Copy all the `bot.js` code  and paste into console.

4. Installed!

5. Now ask for your friend to send you this message: `@HELP`

### Commands:
 @new
 @rules
 @hero [<player> <hero>]
 @config [hero <player> <char>]
 @mate
